# Naviera-Registro

## Description

El objetivo del proyecto es crear una plataforma de gestión de procesos PBIP (Planificación y Control de la Producción en Línea). La plataforma debe ser capaz de gestionar y automatizar las tareas relacionadas con el proceso, como la creación y asignación de tareas, la seguimiento y evaluación del progreso, etc.

Proyecto Naviera-Registro

Naviera-Registro es una plataforma desarrollada para facilitar el proceso de registro de Navieras. A través de esta plataforma, las Navieras pueden registrarse, subir documentos requeridos, generar cotizaciones y realizar un seguimiento del proceso de alta cliente-proveedor.

Especificaciones del Proyecto

Tecnologías Utilizadas:

Backend: Django
Frontend: HTML, CSS, JavaScript
Base de Datos: PostgreSQL
Gestión de Archivos: Amazon S3 o similar
Notificaciones: Integración con servicio de correo electrónico (SendGrid, Mailgun)
Funcionalidades Implementadas:

Registro inicial de clientes
Acceso a documentos clave
Subida y gestión de documentos
Generación y envío de cotizaciones
Seguimiento del proceso de alta cliente-proveedor
